# GS파워 근골격계 테라피 관리 시스템 (Admin Dashboard v2)

## 1. 프로젝트 개요
- **목적:** 서울, 안양, 부천사업소 임직원 대상 테라피 프로그램의 공정한 신청 및 자동 추첨 관리
- **UI/UX 컨셉:** 네오 미니멀리즘(Neo-Minimalism), 시각적 카세트 카드 레이아웃, Apple-like Interaction
- **타겟 유저:** 안전보건팀 관리자 전용 (비밀번호 인증 기반)
- **핵심 가치:** 1. 형평성: 직전 참여자를 시스템에서 자동 필터링하여 기회 균등 제공
  2. 편의성: 직관적인 애플 스타일 UI와 실시간 현황 모니터링
  3. 보안성: 관리자 게이트웨이 적용

## 2. 데이터 및 추첨 로직 (Core Algorithm)
본 시스템은 LocalStorage를 활용하여 브라우저 환경에서 즉시 동작하며, 다음의 4단계 알고리즘을 수행합니다.

1. **상태 감지:** 신청 기한 종료 및 정원(Capacity) 초과 여부 확인
2. **필터링 (1순위):** 신청자 배열 중 `hasPenalty = true`인 직전 회차 참여자를 별도 그룹으로 분리하여 1차 추첨에서 제외
3. **무작위 추첨 (2순위):** 남은 1차 대상자들을 무작위로 섞은(Fisher-Yates Shuffle) 후 정원만큼 추출하여 '최종 선정' 처리
4. **예외 처리:** 1차 대상자가 정원보다 적을 경우 전원 선정 후, 부족한 인원만큼 '제외되었던 직전 참여자 그룹'에서 다시 무작위 추출하여 정원 충족
